## 🚀 Como Executar o Projeto
```bash
# Acesse
https://get-clima.vercel.app

# Ou

# Clone este repositório
$ git clone https://github.com/lucaspanzera1/GetClima.git

# Acesse a pasta do projeto
$ cd GetClima

# Abra o arquivo index.html em seu navegador
```
